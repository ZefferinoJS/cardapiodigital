Estrutura sugerida do projeto

projeto_web/
├── app/
│   ├── config/        # Ficheiros de configuração, como credenciais de DB
│   ├── controllers/   # Ficheiros PHP que contêm a lógica de processamento
│   ├── models/        # Ficheiros PHP que interagem com a base de dados (se houver)
│   ├── views/         # Ficheiros PHP/HTML com o código de apresentação (frontend)
│   └── utils/         # Classes ou funções auxiliares (e.g., validação, formatação)
├── public/
│   ├── css/           # Folhas de estilo (CSS)
│   │   ├── style.css
+│   ├── js/            # Ficheiros JavaScript (JS)
│   │   ├── main.js
+│   ├── images/        # Imagens, ícones, assets visuais
│   └── index.php      # Ponto de entrada único do projeto (Front Controller)
├── vendor/            # Dependências/bibliotecas externas (se usar Composer)
├── .htaccess          # Regras de reescrita de URL (URL Rewriting) e segurança
├── composer.json      # Ficheiro de configuração do Composer (se usado)
└── README.md          # Documentação do projeto
